🔔 แจ้งเตือน

📋 วิธีตั้งค่า LINE (ผ่าน Cloudflare Workers Proxy)
1. สมัคร cloudflare.com ฟรี → Workers & Pages → Create Worker
2. วางโค้ด Worker ด้านล่าง → Deploy → คัดลอก URL
3. ไปที่ developers.line.biz → สร้าง Messaging API → คัดลอก Channel Access Token
4. วาง Worker URL + Token ด้านล่าง → กด ทดสอบ ✅
🔑 LINE + Cloudflare Worker
👥 User ID ผู้เช่า
ผู้เช่าเพิ่มเพื่อน LINE OA แล้วส่งข้อความ → กด "ดึง User ID"
✏️ ตั้งค่าข้อความ
ตัวแปร: {ห้อง} {ผู้เช่า} {วันที่} {จำนวนวัน} {ค่าเช่า} {บัญชี}
📝 โค้ด Cloudflare Worker
คัดลอกโค้ดนี้ไปวางใน Cloudflare Worker Editor แล้วกด Deploy
🧪 ทดสอบและส่ง
📋 วิธีสร้าง Telegram Bot (ฟรี ไม่จำกัด)
1. เปิด Telegram ค้นหา @BotFather พิมพ์ /newbot รับ Bot Token
2. วาง Token ด้านล่าง ค้นหาบอทใน Telegram กด Start ส่งข้อความอะไรก็ได้
3. กด รับ Chat ID กด บันทึก กด ทดสอบส่ง
🔑 Bot Token & เจ้าของ
👥 ผู้เช่า — ลงทะเบียนอัตโนมัติ
ผู้เช่า Start บอท พิมพ์ ชื่อนามสกุล หรือ เบอร์โทร แล้วกดปุ่มด้านล่าง
🧪 ทดสอบและส่งแจ้งเตือน

🔗 ซิงค์ข้อมูลกับ Google Sheets

📋 วิธีตั้งค่า (ทำครั้งเดียว ~10 นาที)
1. ไปที่ script.google.com → New Project → วางโค้ด Apps Script ที่แสดงด้านล่าง
2. กด Deploy → New deployment → Web app
3. Execute as: Me | Who has access: Anyone → Deploy → Copy URL
4. วาง URL ด้านล่าง → กด ทดสอบเชื่อมต่อ → กด โหลดข้อมูล ✅
🔑 Apps Script Web App URL
ยังไม่ได้เชื่อมต่อ
📊 จัดการข้อมูล
☁️ อัปโหลด — ส่งข้อมูลในเครื่องขึ้น Google Sheets (เขียนทับ)
📥 โหลด — ดึงข้อมูลจาก Google Sheets มาใช้ในเครื่องนี้ (เขียนทับ)
📝 โค้ด Apps Script (คัดลอกไปวาง)
คัดลอกโค้ดนี้ไปวางใน script.google.com แล้ว Deploy เป็น Web App

🔥 ซิงค์ข้อมูล Firebase Realtime

📋 วิธีตั้งค่า Firebase (ฟรี Spark Plan)
1. ไปที่ console.firebase.google.com → Create project → ตั้งชื่อ → Continue
2. เมนูซ้าย Build → Firestore Database → Create database → Start in test mode → Enable
3. เมนูซ้าย Project settings (⚙️) → Your apps → กด </> (Web) → Register app
4. คัดลอก firebaseConfig → วางในช่องด้านล่าง → กด บันทึก → ทดสอบเชื่อมต่อ ✅
🔑 Firebase Configuration
ยังไม่ได้เชื่อมต่อ
📊 จัดการข้อมูล
☁️ อัปโหลด — ส่งข้อมูลในเครื่องขึ้น Firebase (เขียนทับ cloud)
📥 โหลด — ดึงข้อมูลจาก Firebase มาใช้ในเครื่องนี้ (เขียนทับ local)
🔄 Realtime mode — ข้อมูลอัปเดตอัตโนมัติทุกอุปกรณ์แบบ live
